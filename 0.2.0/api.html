API Reference¶

Utility function definitions.

cobmo.utils.calculate_irradiation_surfaces(conn, weather_type='singapore_nus', irradiation_model='dirint')¶

Calculates irradiation for surfaces oriented towards east, south, west & north.

  • Operates on the database: Updates according columns in weather_timeseries.
  • Takes irradition_horizontal as measured global horizontal irradiation (ghi).
  • Based on pvlib-python toolbox: https://github.com/pvlib/pvlib-python
cobmo.utils.calculate_sky_temperature(conn, weather_type='singapore_nus')¶

Calculates sky temperatures from ambient air temperature for tropical weather.

  • Ambient air temperature is decreased by 11K to get the sky temperature.
  • TODO: According to ISO ???
cobmo.utils.calculate_error(expected_timeseries=pandas.DataFrame, predicted_timeseries=pandas.DataFrame)¶

Computes the error between expected and predicted timeseries dataframes.

  • Note: This function doesn’t check if the data format is valid.
cobmo.utils.calculate_tank_diameter_height(volume, aspect_ratio)¶

Calculates diameter and height of storage tank based on volume and aspect ratio.

cobmo.utils.get_battery_parameters(battery_parameters_path='/tmp/tmpop0ipz6h/ebb865d70016ba503f05ccfe1ba5bd8ed19a5f4f/data/storage_data/battery_storage_parameters.csv', cost_conversion_factor=1.37)¶

  • Discretization assuming zero order hold

define_augmented_model()¶

Define augmented state space model matrices.

  • Based on Oldewurtel 2011: https://doi.org/10.3929/ethz-a-007157625%0A
  • Construct augmented state space matrices based on ‘normal’ state space matrices.
  • Augmented state space matrices are stored in the build model (not returned).
  • TODO: Validate matrices.
  • TODO: Check alternatives for np.block(), which is slow for a large number of time steps.
simulate(state_initial, control_timeseries, disturbance_timeseries=None)¶

Simulate building model with given initial state and control timeseries.

  • Time horizon is derived from scenario definition in database.
  • Disturbance timeseries is derived from database.
  • TODO: Automatically create initial state from database.
